## What is Slotbot?

Just the project name. An open source tool developed and hosted by [Alf](https://docs.webalf.de/).

## Where is my data stored?

All user data and content is stored in an ISO 27001-certified data centre in Nuremberg, **Germany**.

## Is my data encrypted?

The data is encrypted during all transmissions using the HTTPS protocol.

## What user data is required to use the Slotbot?

To register and start using Slotbot, all you need is a **Discord account**.

## What other third-party services process data?

**No data** from Slotbot is **shared** with third-party systems.
