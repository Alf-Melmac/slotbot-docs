Es existiert bereits ein Event, die ersten Teilnehmer tragen sich ein …

## Eintragen

### Person auf dem Discord

Über den Befehl [Slot erzwingen](/integrationen/discord/bot-befehle/slot-erzwingen) kann eine Person mit der Rolle `Slotbot_Event_Manage` einen Discord-Nutzer eintragen. Der Befehl akzeptiert entweder die Person selbst (`@Name`) oder die Benutzer-ID ([Discord Support Artikel](https://support.discord.com/hc/de/articles/206346498-Wie-finde-ich-meine-Server-ID-), um die Benutzer-ID zu erhalten).

Oder noch besser, der Teilnehmer trägt sich einfach selbst ein [Slot](/integrationen/discord/bot-befehle/slot)!

<Info>
	Für das Umtragen einer Person auf einen anderen Slot muss nur die neue Slotnummer angegeben werden. Der alte Slot
	wird automatisch freigegeben.
</Info>

### Andere Person/Platzhalter/Slot sperren

Über den Befehl [Slot blockieren](/integrationen/discord/bot-befehle/slot-blockieren) kann an einen Slot ein beliebiger Freitext eingetragen werden. Dies verhindert, dass sich ein Interessent auf diesen Platz eintragen kann.

## Austragen

### Person austragen, Platzhalter entfernen, Slot entsperren

Die Belegung eines Platzes in einem Event kann über den Befehl [Unslot erzwingen](/integrationen/discord/bot-befehle/unslot-erzwingen) aufgehoben werden. Dabei spielt es keine Rolle, ob die Belegung eine Person oder nur ein Platzhalter-Text ist.

Jeder Teilnehmer kann sich selbst jederzeit über den [Unslot](/integrationen/discord/bot-befehle/unslot) Befehl austragen.
